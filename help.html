Windows First Aid Toolkit - Help

Windows First Aid Toolkit is a practical AutoIt utility for common Windows first-step repairs. It uses a menu-based interface and a console-style output panel to make common repair tasks faster and easier for technicians and power users.

Main Sections

  • Preparation - Create restore point first.
  • Network repairs - Flush DNS, reset Winsock, release and renew IP.
  • Cleanup - Clear user temp folder, clear system temp folder, clear browser cache for common browsers.
  • Windows repairs - Run sfc /scannow and DISM /Online /Cleanup-Image /RestoreHealth.
  • Service restarts - Restart Windows Update, BITS, and Print Spooler.

Console Output

The bottom black panel is a console-style output area. It shows commands before they run, normal output, warnings, success messages, and section headers.

Usage

  1. Run the app as Administrator.
  2. Leave only the restore point selected, or choose the options you want.
  3. You can also use the Presets menu for quick selection.
  4. Run the selected actions from the Actions menu.
  5. Watch the console output for progress and results.
  6. Save a log copy if needed.

Default Startup Behavior

By default, only the restore point checkbox is selected. All other repair options start unchecked.

Safety Notes

  • SFC and DISM can take a long time.
  • Some actions may require a reboot.
  • Browser cleanup works best when browsers are closed.
  • Locked files may be skipped.
  • Restore point creation depends on System Protection being enabled.
  • This tool is meant for safe first-step repair actions, not deep recovery.

Files Needed Beside the Script or EXE

  • Windows_First_Aid_Toolkit_Help.txt
  • Windows_First_Aid_Toolkit_Help.html

Keep these files in the same folder as the script or compiled EXE so the Help menu items work properly.
